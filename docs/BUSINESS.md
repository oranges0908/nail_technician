# 美甲师能力成长系统 - 业务需求文档

## 产品定位

**美甲师能力成长系统** - 通过AI驱动的美甲师个人能力成长平台

### 核心价值主张

1. **智能设计生成**：基于客户档案（甲型、偏好、历史）+ 灵感图，生成个性化美甲方案
2. **服务后复盘**：AI分析实际完成图vs设计图差异，自动更新客户档案和能力地图
3. **能力成长可视化**：数据驱动的擅长/待提升分析，让成长看得见

### 目标用户

- **主要用户**：独立美甲师、美甲工作室从业者
- **使用场景**：
  - 服务前：为客户生成个性化设计方案
  - 服务中：参考设计方案进行实际操作
  - 服务后：对比分析、复盘总结、能力提升

## 系统业务模块

### 1. 客户管理模块

**业务价值**：建立完整的客户档案，为个性化设计提供基础数据

**核心功能**：
- **客户档案管理**
  - 基本信息：姓名、电话、备注
  - 甲型特征：指甲形状、大小、健康状况
  - 风格偏好：喜欢的颜色、风格、图案类型
  - 禁忌事项：不喜欢的元素、过敏信息

- **甲型标注**
  - 拍照记录客户指甲
  - 标注特殊特征（如甲床宽窄、长度等）

- **偏好记录**
  - 颜色偏好列表
  - 风格标签（简约、华丽、可爱、酷感等）
  - 历史选择分析

**业务流程**：
```
新客户到店 → 创建客户档案 → 拍摄甲型照片 →
记录偏好和禁忌 → 保存档案 → 用于后续设计生成
```

### 2. 智能设计生成模块

**业务价值**：快速生成符合客户偏好的个性化设计方案，提升服务质量和效率

**核心功能**：
- **灵感图库管理**
  - 上传和收藏参考图片
  - 按风格、颜色、季节分类
  - 快速检索和浏览

- **AI设计方案生成**（增强版）
  - 自动读取客户档案信息
  - 结合选择的灵感图
  - **自定义提示词输入**：美甲师可手动输入额外提示词，精确控制设计细节
  - **生成目标选择**：
    - 单指甲设计（用于展示核心图案）
    - 5指设计（单手完整方案）
    - 10指设计（双手配套方案）
  - 支持多方案生成和对比

- **设计微调功能**
  - **智能微调**：选中已生成的设计后，输入调整指令
    - 例如："颜色调亮一点"、"增加金色点缀"、"图案放大"
  - AI根据指令优化设计，保持原设计风格
  - 支持多次迭代微调
  - 保留微调历史记录

- **落地评估**
  - **耗时评估**：
    - 基于设计复杂度自动估算制作时长
    - 考虑因素：图案复杂度、颜色数量、渐变层次、装饰元素
    - 提供时间范围（如：90-120分钟）
  - **用料评估**：
    - 列出所需材料清单（甲油颜色、装饰品等）
    - 估算用量
    - 标注特殊材料（如：金箔、贴纸、钻石等）

- **方案管理**
  - 保存生成的设计方案
  - 关联到特定客户
  - 版本历史记录（包括微调版本）
  - 方案分享和导出

**业务流程**：
```
客户到店 → 选择客户档案 → 与客户沟通需求 →
选择/上传灵感图 → [可选] 输入自定义提示词 →
选择生成目标（1指/5指/10指） → AI生成设计方案 →
[可选] 输入微调指令优化设计 → 查看耗时和用料评估 →
与客户确认方案和价格 → 保存并开始服务
```

### 3. 服务记录模块

**业务价值**：记录完整的服务过程，为后续复盘和能力分析提供数据基础

**核心功能**：
- **服务过程记录**
  - 关联客户和设计方案
  - 记录服务日期和时长
  - 添加服务过程备注

- **实际完成图上传**
  - 拍摄实际完成效果
  - 多角度照片支持
  - 图片质量检查提醒

- **对比展示**
  - 设计图与实际图并排对比
  - 差异高亮显示
  - 支持放大查看细节

**业务流程**：
```
开始服务 → 创建服务记录 → 关联设计方案 →
完成服务 → 拍摄实际完成图 → 上传照片 →
自动触发AI对比分析
```

### 4. 智能复盘模块

**业务价值**：通过AI自动分析，快速发现问题和改进点，加速能力提升

**核心功能**：
- **AI差异分析**
  - 识别设计图与实际图的差异
  - 分析差异的原因（技术、材料、时间等）
  - 生成详细的分析报告

- **自动档案更新**
  - 根据实际效果更新客户偏好
  - 记录客户的真实反馈
  - 优化后续设计生成的准确性

- **能力地图更新**
  - 自动识别本次服务涉及的技能
  - 评估各技能的完成质量
  - 更新能力评分记录

**业务流程**：
```
上传完成图 → AI自动分析差异 →
生成分析报告 → 美甲师查看结果 →
确认客户档案更新 → 能力数据自动记录
```

### 5. 能力分析模块

**业务价值**：可视化展示能力成长轨迹，明确优势和改进方向

**核心功能**：
- **能力雷达图**
  - 多维度能力展示（6-8个维度）
  - 维度包括：颜色搭配、图案精度、细节处理、整体构图、技法运用等
  - 动态更新，反映最新能力水平

- **成长曲线**
  - 时间序列展示各能力维度的变化
  - 对比不同时期的能力水平
  - 识别成长趋势

- **擅长/待提升分析**
  - 基于历史数据自动分析
  - 列出最擅长的3个领域
  - 列出最需要提升的3个领域
  - 提供改进建议

- **智能学习建议**
  - 根据待提升领域推荐学习内容
  - 推荐相关教程和练习方法
  - 设定能力提升目标

**业务流程**：
```
定期查看能力中心 → 查看雷达图和成长曲线 →
了解擅长和待提升领域 → 查看学习建议 →
有针对性地练习和提升
```

### 6. 用户管理模块

**业务价值**：保障账号安全，个性化系统设置

**核心功能**（MVP简化版）：
- **账号管理**
  - 美甲师注册和登录
  - 个人信息设置
  - 密码修改

- **个人设置**
  - 工作室名称和Logo
  - 默认服务时长
  - 通知偏好设置

## 用户故事

### 故事1：新客户首次设计
**角色**：美甲师李美
**场景**：新客户小张第一次来做美甲

**流程**：
1. 李美在系统中创建小张的客户档案，拍摄小张的指甲照片并标注甲型特征
2. 与小张沟通后，了解她喜欢粉色系、简约风格
3. 李美在系统中上传一张小张喜欢的参考图
4. 系统基于小张的档案和参考图，AI生成了3个个性化设计方案
5. 小张选择了其中一个方案，李美开始服务
6. 服务完成后，李美拍摄实际完成图上传
7. 系统AI分析显示：颜色还原度95%，图案精度90%，整体相似度92%
8. 系统自动记录李美在"粉色系配色"和"简约风格"上的能力表现

**价值**：
- 快速生成符合新客户偏好的设计
- 客观评估服务质量
- 自动积累能力数据

### 故事2：定期能力复盘
**角色**：美甲师李美
**场景**：做了50单服务后，想了解自己的能力成长

**流程**：
1. 李美打开能力中心，查看能力雷达图
2. 发现"颜色搭配"和"简约风格"得分最高（90+分）
3. 但"复杂图案绘制"和"渐变技巧"得分较低（60-70分）
4. 查看成长曲线，发现"颜色搭配"能力最近一个月提升了15%
5. 系统建议：多练习花卉图案和渐变技巧，推荐相关教程

**价值**：
- 清晰了解自己的优势和短板
- 数据驱动的成长决策
- 有针对性的学习和练习

### 故事3：老客户回访设计
**角色**：美甲师李美
**场景**：客户小张第5次来做美甲

**流程**：
1. 李美选择小张的档案，系统自动加载小张的历史偏好
2. 系统显示：小张过去4次都选择了粉色系，3次选择了简约风格，1次选择了可爱风
3. 李美上传一张新的灵感图，系统生成的方案自动融合了小张的偏好
4. 生成的方案非常符合小张的口味，直接确认使用

**价值**：
- 客户档案持续优化，越用越精准
- 节省与老客户的沟通时间
- 提升客户满意度和忠诚度

### 故事4：设计微调和评估（新功能）
**角色**：美甲师李美
**场景**：为客户生成了设计方案，但客户希望微调细节

**流程**：
1. 李美为客户生成了一个10指双手的设计方案
2. 客户喜欢整体风格，但觉得颜色太深
3. 李美在系统中输入微调指令："颜色调亮20%，保持图案不变"
4. AI生成新版本，颜色变亮但图案完全保留
5. 客户满意后，李美查看落地评估：
   - 预估耗时：100-120分钟
   - 用料清单：浅粉色甲油、白色甲油、金箔、封层
6. 李美根据评估与客户确认价格和预约时长

**价值**：
- 快速响应客户的微调需求，无需重新生成
- 准确的耗时评估，避免超时影响后续预约
- 用料清单帮助提前准备材料，提升服务流畅度
- 基于评估的精准定价，提升收益

## 业务指标

### 核心指标

**效率指标**：
- 设计方案生成时间：< 1分钟/方案
- 客户满意度：目标 > 90%
- 复盘分析时间：< 30秒/次

**成长指标**：
- 能力提升趋势：每月至少1个维度提升5%
- 服务质量分数：目标平均 > 85分
- 设计vs实际相似度：目标 > 80%

**使用指标**：
- 客户档案完整度：目标 > 90%
- 服务记录完成率：目标 100%
- 复盘功能使用率：目标 > 80%

## MVP范围定义

1. **客户管理**：创建、编辑、查看客户档案，上传甲型照片，记录偏好
2. **AI设计生成（增强版）**：
   - 上传灵感图，生成个性化设计方案

   - **生成目标选择**（1指/5指/10指）

5. **能力雷达图**：展示6个能力维度的评分

5. 社区分享功能
6. 多美甲师协作
7. 客户端小程序
8. 智能定价建议（基于复杂度自动推荐价格）

## 风险和挑战

### 业务风险

1. **AI生成质量**：AI生成的设计可能不符合预期
   - 应对：提供多方案选择，支持重新生成，收集反馈持续优化prompt

2. **用户习惯培养**：美甲师可能不习惯使用数字化工具
   - 应对：简化操作流程，提供新手引导，强调效率提升

3. **数据隐私**：客户档案涉及个人信息
   - 应对：遵守数据保护法规，提供数据加密，明确隐私政策

### 技术挑战

1. **AI成本控制**：OpenAI API调用成本较高
   - 应对：优化prompt减少token消耗，考虑缓存机制，设置每日额度

2. **图像处理性能**：移动端上传和处理大图片
   - 应对：前端压缩图片，后端异步处理，提供进度提示

3. **能力模型准确性**：如何科学评估美甲师能力
   - 应对：MVP阶段使用简化模型，后续结合专家经验优化算法
